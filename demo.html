Foundation is simply some CSS and java files that give you classes to create better web page. The essential part of foundation is the grid system. A div can become a row, and sub-divs can become columns of that row. Each page is a maximum of 12 columns long. The columns resize with the window and align based on how you set them. This allows for sites to look good no matter how large the browser is.
